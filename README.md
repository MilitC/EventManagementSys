Event Management System: A comprehensive, full-stack web application designed to streamline the event planning lifecycle. This system allows organizers to create and manage events while providing attendees with a seamless interface to view and register for upcoming activities.
View Live application: https://event-management-sys-dun.vercel.app/

Role-Based Management: Secure authentication system for organizers and attendees.
Event Lifecycle Management: Full CRUD (Create, Read, Update, Delete) functionality for events.
Dynamic Filtering: Easily browse events by category, date, or popularity.
Responsive UI: Mobile-friendly design optimized for all screen sizes.
Secure API: Robust backend communication for data integrity.
